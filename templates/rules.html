Вас приветствует стартовая страничка блог-платформы Big-Blog

Для того чтобы начать использовать git-blog необходимо:

  • Зарегистрироваться в нашей системе:
    • ввести существующее GitHub username
    • ввести имя существующего репозитория в котором есть папка posts
    • а также ввести пароль по которому вы сможете редактировать свои посты
  • Внести в папку posts файлы Вашего блога в формате необходимом для корректной работы Big-Blog
    • Файла должен быть в формате Markdown
    • Имя файла должно соблюдать формат: 'YYYY-MM-DD-some-post-title.md'
    • Содержание файла должно быть следующим:
        например
      • ---(область заголовков отделена этими символами)
      • title: some_title (заглавие поста)
      • tags: some_tag (тема или темы через запятую)
      • author: some_author (имя автора)
      • date: YY-MM-DD HH:MM (формат даты соблюдаем в таком виде для правильного отображения)
      • layout:some_layout (шаблон для отображения постов, можно оставить пустым)
      • ---(область заголовков отделена этими символами)
      • Ну и текст поста
  • Для того что-бы изменения отображались в вашем блоге каждый раз когда вы делаете изменения:
      Необходимо настроить web-hook:
    • своем репозитории зайти в настройки
    • затем выбрать add webhook
    • Payload url вставить http://gitblog.pythonanywhere.com/git_username/git_rep_name/web_hook (в моем случае это http://gitblog.pythonanywhere.com/rrlero/git-blog/web_hook)
    • в content type выбрать application/json
    • затем галочку на Just the push event и Active
    • финальный аккорд Update Webhook
